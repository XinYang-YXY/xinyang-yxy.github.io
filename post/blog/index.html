public static List<int> CheckStock(List<Product> ps)
        {
            var query = from s in ps
                        where s.Qty <= 10
                        select s.Id;
            var toBeRestock = query.ToList();
            return toBeRestock;
        }

Qn2

  • Advantages
    • Combined with ORM framework such as EntityFramework, it allows a type-safe encoding of database query
    • C# compiler checks for syntax errors of query
    • C# compiler checks for semantic errors of the query
      • For example, relational property among models.
      • SQL errors become type errors.
    • The support for NoSQL database comes nearly free.
    • Database migration does not affect code.
    • SQL injection is handled automatically
  • Disadvantages
    • It has a stiff learning curve
    • It is a bit unnatural to express certain queries, such as join and groupby.
