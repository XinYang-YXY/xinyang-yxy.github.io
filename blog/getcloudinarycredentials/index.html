Quick guide to obtain your cloudinary credentials

  • Go to Cloudinary site
  • Obtain the credentials

Go to Cloudinary site

  1. Open the Cloudinary URL into your browser.
  2. Register an account and login to the dashboard

Obtain the credentials

  1. In the dashboard, under the ‘Account Details’. You should be able to find your cloudinary credentials. It should look something like the following
Cloud name:	ugdhfygwegu
API Key:	1754611353467 Copy to clipboard
API Secret:	*************************** Copy to clipboard Reveal
API Environment variable: CLOUDINARY_URL=cloudinary://***************:*************************** Copy to clipboard Reveal
  1. DONE!
